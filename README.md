# Workast for Zoho Cliq

Developed a Workast task-management extension integrated with Zoho Cliq
Enabled users to create and manage tasks, notes, and spaces with real-time updates
Implemented channel mapping, task comments, message-to-task conversion, and smart automation
Built an interactive Workast Widget with Dashboard, Tasks, Deadlines, Team, and Notes views
Added a daily reminder scheduler to notify users of tasks due today

---

## Overview

This extension connects Zoho Cliq with Workast and provides tools to create, manage, and monitor tasks from chat. Users can view tasks, deadlines, team information, and notes using an interactive widget. It also supports automation features such as message-to-task conversion and daily task reminders.

---

## Features

- Workast account linking  
- Create, view, update, and delete tasks  
- Manage Workast spaces and notes  
- Channel-to-space mapping  
- Add comments to tasks  
- Convert Cliq messages into Workast tasks  
- Interactive widget with:
  - Dashboard  
  - Tasks  
  - Deadlines  
  - Team  
  - Notes  
- Daily scheduler to notify users of tasks due today  

---

## AI-Based Task Automation

The extension supports AI-assisted commands to simplify task management.

- Convert normal chat messages into structured tasks  
- Create and assign tasks using natural language  
- Update or delete tasks using AI commands  
- Get summaries of today’s tasks and overdue items  
- Generate space-level task insights  

---

## Workast Widget

The Workast widget provides a clean interface inside Zoho Cliq:

- **Dashboard** – Overview of tasks and activity  
- **Tasks** – View tasks across Workast spaces  
- **Deadlines** – Upcoming and overdue tasks  
- **Team** – Workspace team members  
- **Notes** – Notes across all spaces  

All views are dynamically loaded within Cliq.

## Installation

### Step 1: Login or create a Zoho Cliq account
https://cliq.zoho.com/

### Step 2: Install the extension
https://cliq.zoho.com/installapp.do?id=8267

### Step 3: Link your Workast account
- Open Zoho Cliq  
- Select **Link Workast Account**  
- Enter your Workast API details  

### Step 4: Start using the extension
- Use `/workast` commands  
- Open the Workast widget  
- Map channels to spaces  
- Enable daily reminders
